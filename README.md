# Манка-Запеканка

Демонстрационный сайт кофейни — портфолио-проект. Статическая вёрстка (HTML/CSS/JS), без бэкенда.

Кофейня вымышленная, сайт создан как концепт для портфолио.

## Структура

- `index.html` — главная страница
- `menu.html` — полное меню
- `privacy.html` — политика обработки персональных данных (для формы бронирования)
- `assets/` — изображения, видео, favicon

## Деплой

Сайт статический, хостится на GitHub Pages.
